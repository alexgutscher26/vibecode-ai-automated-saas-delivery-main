Develop a comprehensive Idea Intake Module for SaaS product management with the following specifications:

1. Basic Information Capture:
   - Required fields:
     * Idea Name (text input, max 100 characters)
     * Description (rich text editor with formatting options, min 50 characters)

2. Advanced Mode (toggle-enabled section):
   - Target Persona:
     * Demographic information (age range, job title, industry)
     * Behavioral characteristics (usage patterns, decision-making criteria)
   - Pain Points:
     * Current challenges (bulleted list with add/remove functionality)
     * Impact assessment (severity scale 1-5)
   - Competitive Differentiation:
     * Unique value proposition (text area, 200 characters max)
     * Comparison matrix with existing solutions
   - Pricing Hypothesis:
     * Pricing model options (dropdown: subscription, freemium, pay-per-use)
     * Price range sliders (min/max values)
     * Willingness-to-pay validation questions

3. Technical Requirements:
   - Responsive design for desktop and mobile
   - Form validation with clear error messages
   - Auto-save functionality every 30 seconds
   - Export options (PDF, CSV formats)

4. Success Criteria:
   - 95% form completion rate for basic mode
   - 60% engagement with advanced mode features
   - Submission processing time under 2 seconds
   - Accessibility compliance (WCAG 2.1 AA)
